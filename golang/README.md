# 历史

为了解决在21世纪多核和网络化环境下越来越复杂的编程问题, Ken Thompson(UNIX操作系统、B语言作者) 和 Rob Pike 以及 Robert Griesemer(设计了 v8 引擎和 HotSpot 虚拟机)合作发明了 go 语言。

go 的基础语法源自于 c，模块导入概念 Oberon，goroutine 的概念则借鉴于 Alef。
